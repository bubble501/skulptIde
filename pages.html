Skulptide

An online Python development environment.

Download .zip Download .tar.gz View on GitHub

The Skulpt IDE is my attempt to build an online Integrated Development Environment where anyone can edit Python code and see it running on the the browser. It uses the code base for Skulpt, a JavaScript port of the Python language, the CodeMirror online code editor and several other useful modules.

Installing

Just download the whole thing (use the download .zip link in this page) into some folder your web server can see and access it. Then access the index.html page from there and you are done.

In fact, you don't even need to download it to your site, but use it directly from the github servers, by accessing https://rawgit.com/esperanc/skulptIde/master/index.html.
